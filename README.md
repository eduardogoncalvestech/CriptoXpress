# 🔐 CriptoXpress

**CriptoXpress** é uma ferramenta leve e intuitiva para criptografar e descriptografar textos com segurança, usando apenas **HTML, CSS, JavaScript e Python puro** — sem frameworks, sem dependências externas no frontend, e com criptografia forte no terminal.

---

## 🚀 Funcionalidades

- ✅ Criptografia leve via navegador (base64 com chave)
- 🔐 Criptografia forte via terminal (AES com chave secreta)
- 🧠 Descriptografia com validação de chave
- 💻 Interface web responsiva e fácil de usar
- 📦 Código limpo, modular e pronto para evoluir
